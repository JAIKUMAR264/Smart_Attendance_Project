\# Smart Attendance Project



\## Overview



Smart Attendance Project is a Python-based attendance management system that helps track student attendance and generate attendance reports automatically.



\## Features



\* Mark student attendance

\* Store attendance records in CSV format

\* Generate attendance reports

\* Easy to use and lightweight

\* Simple file-based storage



\## Project Structure



```text

Smart\_Attendance\_Project/

│

├── attendance.py

├── attendance\_data.csv

├── attendance\_report.csv

├── requirements.txt

└── README.md

```



\## Requirements



\* Python 3.x



Install dependencies:



```bash

pip install -r requirements.txt

```



\## How to Run



```bash

python attendance.py

```



\## Output Files



\### attendance\_data.csv



Stores the attendance records.



\### attendance\_report.csv



Contains the generated attendance report.
